# ¿Qué es AVStack?
AVStack es el nombre comercial de AV Works System, un equipo de desarrollo
de software con sede en Sevilla (CP 41000), Andalucía, España.

Nos especializamos en:
- Desarrollo web a medida: sitios corporativos, e-commerce, landing pages
- Aplicaciones Android nativas con Kotlin y Jetpack Compose
- Progressive Web Apps (PWA)
- Backend y APIs REST/GraphQL con Node.js
- Seguridad web, auditorías y pentesting
- Diseño de marca e identidad visual
